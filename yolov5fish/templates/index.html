WHAT IS THE PROJECT ABOUT?

There is an ongoing communication barrier between the deaf and the rest of the community. Text writing and interpreters are their only ways of communications. The inability of the two groups to communicate effectively has an impact on their interpersonal relationships. Current study is to develop a deep learning based image classification architecture, which recognizes Pakistan Sign Language Urdu alphabets signs and transforms them into textual form. The key feature of this research is that there will be no need to hire an interpreter to translate finger spellings between the two. Deep learning architecture YOLO “You Only Look Once” is a widely used algorithm for object detection. Using the modified architecture we are able to take image input of hand signs and classified what Urdu huroof-e-tahajji they correspond to, in textual form. Further, there was a need to create our own dataset, which consists of approximately 23000 images of Urdu huroof-e-tahajji and with the help of augmentation, the dataset is now more generalized, hence, it will make the model’s performance more robust. Subsequently, current research is twofold, as it gives an accurate and efficient model for Urdu sign language recognition as well as a newly developed dataset.
